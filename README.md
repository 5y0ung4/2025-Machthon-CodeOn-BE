  ### 컨벤션
| type | 설명 |
| ---------- | ---------------------------------------------- |
|feat|새로운 기능 추가 (Feature)|
|fix|버그 수정 (Bug Fix)|
|docs|문서(README, 주석 등) 수정|
|style|코드 포맷팅, 세미콜론 누락 등 (코드 로직 변경 없음)|
|refactor|코드를 리팩토링(재구조화)하여 개선 (기능 변경 없음)|
|test|테스트 코드 추가 또는 수정|
|chore|빌드 시스템, 라이브러리, 설정 파일 변경 등 (프로젝트 관리)|
|build|빌드 관련 파일(gradle, maven 등) 변경|
